Cuando se crea un blog, la mayoría de las aplicaciones pueden contener las siguientes listas, bibliotecas y funciones para ayudarte a crear contenido.



1

Entradas. Las entradas son una parte esencial de un blog y donde se comunica el autor con los lectores mediante entradas con fecha

2

Comentarios. Obtén opiniones sobre las entradas de blog mediante comentarios de los lectores. Puedes ver los comentarios de un lector viendo cada entrada o puedes ver todos los comentarios en el blog viendo la lista comentarios predeterminada.

3

Categorías. Usa las categorías para organizar las entradas, de forma que los lectores puedan encontrar fácilmente las entradas que más les interesen.

4

Lista a otros blog. Usa esta lista para guardar vínculos a otros blog, como blog que cubran asuntos similares, de compañeros o que hagan referencia a su blog.

5

Vínculos. Usa la lista vínculos para guardar otros tipos de vínculos, como sitios Web relacionados o información de fondo.

6

Fotografías. La lista de fotografías es una biblioteca de imágenes donde puedes guardar fotos de interés y vincularlas a las entradas de blog.

7

Archivos para entradas. Las entradas se archivan por día y por mes, de forma que los lectores puedan buscar entradas basadas en determinadas fechas. Un blog contiene dos vistas predeterminadas del calendario: una vista muestra las entradas en una lista estándar y la otra muestra las entradas en una vista de calendario.

8

Fuentes RSS. El autor o el lector puede permanecer actualizado sobre los blog suscribiéndose a fuentes RSS.
