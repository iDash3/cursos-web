Tema 1 ¿Qué es la Web 2.0?

Repasemos algunos conceptos básicos sobre "El Internet".

Definición

Internet es una enorme red global compuesta de la conexión de varias computadoras que intercambian información entre ellas.

Origen

Internet empezó a usarse más, aproximadamente en 1994, sin embargo antes ya se utilizaba para entender necesidades de investigación y comunicación.

Otros Nombres

Internet recibe varios nombres, algunos son:

  • “La Red”
  • “La Supercarretera de Información”
  • “La Red Mundial de Información”
  • “El Ciberespacio”
Evolución

El origen del termino Web 2.0 es asociado usualmente con Tim O’Reilly debido a la referencia hecha en la conferencia O’Really Media Web 2.0 en el año 2004 y fue utilizado para referirse a una segunda generación en la historia del desarrollo de tecnología Web basada en comunidades de usuarios y una gama especial de servicios, como las redes sociales, los blogs o las Wikis, que fomentan la colaboración y el intercambio ágil y eficaz de información entre los usuarios de una comunidad o red social.
